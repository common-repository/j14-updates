== Description ==
Official widget for j14.org.il, the Israeli Tent Movement for social justice.
Calls blog viewers to support the struggle via three links to j14.org.il:

1. Get Updated - Sign up for for weekly newsletter and/or immediate updates
2. Volunteer - Fill in a volunteer form
3. Donate - Donate money via official channels.
The widget will smartly resize itself to fit in your sidebar. Minimum width: 100px.

== Installation ==

1. Download the file 'j14-updates.zip'
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets and drag-drop the widget to the relevant widget area
Manually:

1. Upload `j14-updates` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets and drag-drop the widget to the relevant widget area
